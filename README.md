# Bright Future Website

Static Vercel-ready website.

Important: external CSS/JS/images are intentionally loaded from `https://upscrolled.com` to preserve the downloaded Elementor style. The main phone frame is local at `assets/app_frame.png`.

Deploy:
1. Push these files to GitHub.
2. Import repo in Vercel.
3. Framework preset: Other.
4. Build command: empty.
5. Output directory: empty/root.
